# 更新日志

本文档记录 Codex Switcher 面向用户的重要变化。开发中的内容维护在“未发布”章节，发布时由脚本转换为对应版本和日期。

英文版本维护在 [CHANGELOG.en.md](./CHANGELOG.en.md)，发布时两份日志必须保持版本同步。

## [未发布]

### 新增

- 更新提示支持展示本次版本重点，并可跳转查看完整更新记录。
- 更新说明支持中英文，英文环境会显示英文重点并打开英文完整日志。

### 优化

- 统一应用与任务栏托盘图标，左键点击托盘图标直接打开主窗口。
- 重新整理托盘右键菜单，集中展示当前账户、快捷操作和显示组件。
- 使用原生 Windows 进程检测，并只跟踪实际的 Codex 桌面进程，减少切换前检查阻塞。
- 完善发布工具，严格校验版本、双语 Changelog、签名和跨平台更新清单。

### 修复

- 修复 Windows 进程检测堆积导致账户切换卡死的问题。
- 修复取消账户切换后托盘菜单出现多个对勾的问题。
- 修复托盘连续发起账户切换时状态错乱或重复处理的问题。
- 修复进程检查失败后界面继续显示旧状态，以及单账户刷新失败没有明确提示的问题。

## [0.105.0] - 2026-07-23

### 新增

- 主页面支持列表和卡片两种账户布局。
- 支持记忆主窗口尺寸、位置和最大化状态。

### 优化

- 调整标题栏、账户操作区和布局切换按钮的排列。
- 卡片布局直接融合账户详情信息，减少额外跳转。
- 优化账户筛选、订阅到期和 API 账户相关文案。
