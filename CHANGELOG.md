# 更新日志

本文档记录 Codex Switcher 面向用户的重要变化。开发中的内容维护在“未发布”章节，发布时由脚本转换为对应版本和日期。

## [未发布]

### 新增

- 更新提示支持展示本次版本重点，并可跳转查看完整更新记录。

### 优化

- 统一应用与任务栏托盘图标，左键点击托盘图标直接打开主窗口。
- 重新整理托盘右键菜单，集中展示当前账户、快捷操作和显示组件。

### 修复

- 修复 Windows 进程检测堆积导致账户切换卡死的问题。
- 修复取消账户切换后托盘菜单出现多个对勾的问题。

## [0.105.0] - 2026-07-23

### 新增

- 主页面支持列表和卡片两种账户布局。
- 支持记忆主窗口尺寸、位置和最大化状态。

### 优化

- 调整标题栏、账户操作区和布局切换按钮的排列。
- 卡片布局直接融合账户详情信息，减少额外跳转。
- 优化账户筛选、订阅到期和 API 账户相关文案。
